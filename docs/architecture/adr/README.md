# Architecture Decision Records

ADRs capture significant, durable choices and their consequences. Use the next four-digit number, a short kebab-case title, and the sections: status, context, decision, consequences, and alternatives considered. Accepted ADRs are immutable except for status/links; superseding decisions get a new ADR.
